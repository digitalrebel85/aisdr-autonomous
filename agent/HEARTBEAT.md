# Heartbeat Checklist - Autonomous SDR

When you wake up, check these (rotate through, don't do all every time):

## Priority 1: Always Check
- [ ] **Email Replies** - Check Nylas for new replies, categorize and process
- [ ] **Meetings Booked** - Check calendar for new bookings, notify Chris
- [ ] **Bounces/Errors** - Check for delivery issues, pause if needed

## Priority 2: Campaign Management  
- [ ] **Send Queue** - Generate and queue emails for today's sends
- [ ] **Reply Follow-ups** - Schedule appropriate follow-up sequences
- [ ] **Campaign Performance** - Review metrics, flag underperformers

## Priority 3: Lead Pipeline
- [ ] **Apollo Search** - Discover new leads matching ICP
- [ ] **Lead Enrichment** - Enrich discovered leads with company data
- [ ] **Lead Scoring** - Score leads, prioritize high-value targets

## Priority 4: Optimization
- [ ] **Email Templates** - Review low-performing copy, suggest rewrites
- [ ] **Subject Line Tests** - Analyze A/B test results
- [ ] **ICP Refinement** - Review which prospects respond best

## Priority 5: Reporting
- [ ] **Mission Control Update** - Refresh dashboard data
- [ ] **Daily Summary** - Log activity to memory file
- [ ] **Chris Update** - Send brief status if significant events

## Safety Checks
- [ ] Send limits not exceeded?
- [ ] No spam complaints?
- [ ] Email reputation healthy?
- [ ] All API keys working?

## If Nothing Needs Attention
Reply: HEARTBEAT_OK and go back to sleep.

## If Issues Found
Message Chris immediately with:
- What the issue is
- Impact (emails paused, etc.)
- Suggested fix
- Need approval for: [specific action]
